# 15-Day Python for AWS/DevOps

This workspace is your Python learning track.

Current status:

- `python_15_day_plan/day01_lists.py` is your Day 1 basics file.
- `python_15_day_plan/` contains Day 1 through Day 15 lesson files.
- `final_project/` contains the DevOps-style project scaffold.
- `DAILY_TASKS.md` shows exactly where you are and what to do each day.

Daily rhythm:

1. Read the lesson comments.
2. Run the file.
3. Change the TODO section yourself.
4. Test at least two cases.
5. Add one short comment about what you learned.

Run a lesson:

```bash
python3 python_15_day_plan/day02_lists.py
```

Run the final project:

```bash
python3 final_project/devops_tool.py
```

Check the course files:

```bash
python3 check_course.py
```

Check your daily Git status:

```bash
python3 scripts/daily_git_check.py
```
