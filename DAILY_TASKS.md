## Daily Rules

Each study day is 2 hours:

1. 30 minutes: read and understand the lesson comments.
2. 45 minutes: run the example code and change values.
3. 30 minutes: complete the homework yourself.
4. 15 minutes: test, fix mistakes, and add one comment about what you learned.

Each day is complete only when:

- the lesson file runs without errors
- the homework section is completed
- at least 2 test cases are tried
- you can explain the main idea in your own words
